# Homework 10.1
Данный проект реализует такие функции как:
- Получение маски карты в формате XXXX XX** **** XXXX, где X — это цифра номера
- Получение маски карты в формате **XXXX, где X — это цифра номера.
- Функция возвращает тип карты и ее маску
- Возвращает дату в привычном для человека виде
- Возвращает список по состоянию
- Функция возвращает список на убывание по дате
